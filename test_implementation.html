üêõ Troubleshooting

If modals don't appear above topbar:

  • Check that topbar has no fixed positioning
  • Verify modal z-index is set to 9999
  • Ensure modal position is fixed

If dynamic forms don't work:

  • Check browser console for JavaScript errors
  • Verify event delegation is working
  • Check that field names are properly indexed

If AJAX submission fails:

  • Check CSRF token is included
  • Verify API routes are registered
  • Check browser network tab for request details
